techniques

Techniques describe the means by which adversaries achieve tactical goals. They represent “how” an adversary achieves a tactical objective by performing an action.

Techniques may also represent “what” an adversary gains by performing an action.

There can be multiple techniques in each tactic category as there are many ways to achieve tactical objectives. [1]
